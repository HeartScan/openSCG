# Castiglioni 2022: SCG for Sleep Monitoring

> **Quick Conclusion:** Validates the use of mechanocardiography for long-term physiological monitoring during sleep.

---

### 📊 Key Accuracy Metrics

| Metric | Result |
| :--- | :--- |
| **Finding** | SCG can track autonomic changes during sleep stages |

---

### 🔍 Study Analysis

#### **Objective & Population**
Observational Sleep Study. Cohort: Healthy and sleep-disordered subjects (N=10).

#### **What it Supports**
The study supports using SCG for non-invasive, overnight monitoring of heart activity without the need for electrode patches.
